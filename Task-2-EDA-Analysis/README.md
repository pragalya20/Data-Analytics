# Sales EDA & Business Intelligence Project

## Overview
This project focuses on performing Exploratory Data Analysis (EDA) on a sales dataset to uncover patterns, trends, and business insights. It also includes SQL queries and a dashboard for visualization.

---

## Objectives
- Analyze sales performance across products, categories, and regions
- Identify top-performing products
- Understand monthly sales trends
- Build a dashboard for business insights

---

## Dataset
The dataset contains 240 records with the following features:
- Transaction ID
- Date
- Product Category
- Product Name
- Units Sold
- Unit Price
- Total Revenue
- Region
- Payment Method

---

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL
- Excel (Dashboard)

---

## Key Analysis Performed
- Descriptive statistics
- Univariate analysis
- Business question analysis
- Correlation analysis
- Revenue validation

---

## Key Insights
- Electronics category generates the highest revenue
- North America contributes the most sales
- Sales show a declining trend after initial months
- Top products include MacBook Pro and Canon Camera
- Revenue strongly depends on units sold

---

## SQL Queries
SQL queries were used to extract insights such as:
- Top 5 products by revenue
- Revenue by category and region
- Monthly sales trends
- Payment method usage

---

## Conclusion
This project demonstrates how data analysis and visualization can provide actionable business insights and support decision-making.
